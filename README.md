# Portfolio

Minimal personal portfolio site for Lucas da Silva.

**Live site:** https://lucasdasilva8.github.io/portfolio/

## Projects featured

- [SketchForge](https://lucasdasilva8.github.io/sketchforge/) — sketch to parametric 3D CAD
- [Melanoma Detection](https://lucasdasilva8.github.io/melanoma-detector/) — AI skin lesion screening

## Local preview

Open `index.html` in a browser, or serve locally:

```bash
python3 -m http.server 8080
```

Then visit http://localhost:8080

## Deploy to GitHub Pages

1. Push this repo to `lucasdasilva8/portfolio` on GitHub
2. Go to **Settings → Pages**
3. Set source to **Deploy from branch `main` / root**
4. Site will be live at https://lucasdasilva8.github.io/portfolio/

No build step required — static HTML and CSS only.
